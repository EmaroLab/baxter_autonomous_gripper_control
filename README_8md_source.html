17 ## The project
18 
19 This project has the main goal in designing a ROS node for the Baxter Research Robot, to let the robot predict gripper position during a 'pick and place' task.
20 For the purpose, inertial data, from a smartwatch and Baxter's joints, has been collected and combined to fit an LSTM recurrent neural network.
21 
22 #### In this repository:
23 * **_src_** folder
24  * record_data.py &nbsp;:arrow_right:&nbsp; _ROS node to collect data coming from the smartwatch and Baxter_
25  * auto_grasping.py &nbsp;:arrow_right:&nbsp; _ROS node to predict in real time the gripper position_
26 * **_scripts_** folder
27  * RawData_Interpolate.py &nbsp;:arrow_right:&nbsp; _Python scripts to interpolate Baxter data with smartwatch timesteps_

35 #### Step1: Autonomous grasping
36 **_Currently_**, this study has reached the goal of the autonomous grasping with the following conclusions:
37  * The Baxter grasping action does not depend on the position of the object on a flat surface
38  * The Baxter grasping action can be predicted using objects of different shape associated with different ways of grabbing (e.g. a glass grabbed from the side or a ball grabbed from above)
39  * It is not possible, with used technologies, to discriminate between the grabbing action and the release action without manipulating raw data such that: every time sequence of the dataset take into account the initial state of the gripper for that sequence.
