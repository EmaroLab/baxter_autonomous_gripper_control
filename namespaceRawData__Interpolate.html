Detailed Description

This script generates sequences to fit a recurrent neural network, starting from separated files of Baxter and smartwatch raw data.

For each sequence it interpolates Baxter data using the smartwatch timestamps. In this version, the final result does not consider timestamps as variable for the final dataset.
