Detailed Description

This is a ROS node to collect data simultaneusly from baxter limb and gripprer (left) and a smartwatch.

It subscribes to the "/imu_giver" topic for the smartwatch data and it uses Baxter interface for the Baxter data. It can record sequences using stop (s) and new (n) commands.

It write data for each sequence as follows:

  • A csv file for Baxter data
  • A csv file for smartwatch data Every row in both files has a timestamp IN NANOSECONDS as last element, useful to join data coming from the two different devices.
